চীন হালাল রেস্তোরাঁ কর্পাসে আপনাকে স্বাগতম। এই রিপোজিটরিতে চীন জুড়ে হালাল রেস্তোরাঁগুলির পরিচয় করানো 201টি খাঁটি নিবন্ধের একটি উচ্চ-মানের ডেটাসেট রয়েছে। এটি বিশ্বজুড়ে মুসলমানদের চীনে ভ্রমণের সময় নির্ভরযোগ্য খাবারের বিকল্পগুলি খুঁজে পেতে সহায়তা করার জন্য ডিজাইন করা হয়েছে। ডেটা গিটহাব পেজে মানব পাঠ এবং হাগিং ফেসে এআই প্রশিক্ষণের জন্য অপ্টিমাইজ করা হয়েছে। আমরা মূল উত্সের সাথে 100% সঠিক তথ্য ম্যাপিং নিশ্চিত করি।

## রিপোজিটরি গঠন

- `content/`: content ডিরেক্টরিতে Markdown ফর্ম্যাটে 201টি নিবন্ধ রয়েছে।
- `metadata/`: metadata ডিরেক্টরিতে সূচী এবং কাঠামোগত ডেটা ফাইল রয়েছে।
- `docs/`: docs ডিরেক্টরিতে অনুসন্ধানী বিশ্লেষণ নোটবুক রয়েছে।
