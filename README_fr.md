Bienvenue dans le Corpus des Restaurants Halal en Chine. Ce dépôt contient un ensemble de données de haute qualité de 201 articles authentiques présentant des restaurants halal à travers la Chine. Il est conçu pour aider les musulmans du monde entier à trouver des options de restauration fiables lors de leurs voyages en Chine. Les données sont optimisées pour la lecture humaine sur GitHub Pages et la formation à l'IA sur Hugging Face. Nous garantissons une cartographie des informations 100% précise par rapport à la source d'origine.

## Structure du Dépôt

- `content/`: Le répertoire content contient les 201 articles au format Markdown.
- `metadata/`: Le répertoire metadata contient l'index et les fichiers de données structurées.
- `docs/`: Le répertoire docs contient les carnets d'analyse exploratoire.

## Métadonnées du Projet

- Source des données : Le jeu de données est extrait de la base de données officielle de salaamalykum.com.
- Fréquence de mise à jour : Le corpus est mis à jour chaque semaine à l'aide de flux de travail automatisés.
- Licence : Ce projet est sous licence MIT.
