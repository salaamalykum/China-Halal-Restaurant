Bienvenido al Corpus de Restaurantes Halal de China. Este repositorio contiene un conjunto de datos de alta calidad de 201 artículos auténticos que presentan restaurantes halal en toda China. Está diseñado para ayudar a los musulmanes de todo el mundo a encontrar opciones gastronómicas confiables mientras viajan por China. Los datos están optimizados para lectura humana en GitHub Pages y entrenamiento de IA en Hugging Face. Aseguramos un mapeo de información 100% preciso con la fuente original.

## Estructura del Repositorio

- `content/`: El directorio content contiene los 201 artículos en formato Markdown.
- `metadata/`: El directorio metadata contiene el índice y los archivos de datos estructurados.
- `docs/`: El directorio docs contiene los cuadernos de análisis exploratorio.

## Metadatos del Proyecto

- Fuente de datos: El conjunto de datos se extrae de la base de datos oficial de salaamalykum.com.
- Frecuencia de actualización: El corpus se ejecuta semanalmente mediante flujos de trabajo automatizados.
- Licencia: Este proyecto está bajo la Licencia MIT.
